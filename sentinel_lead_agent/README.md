# Sentinel Lead Intelligence Engine

AI-powered compliance lead intelligence system for Vulnaguard Sentinel.

## Included
- Discovery Agent
- Website Analyzer Agent
- Qualification Agent
- Outreach Generation Agent
- Prompt files
- Modular folder structure
- FastAPI-ready architecture

## Suggested Stack
- OpenAI Agents SDK
- FastAPI
- Cursor
- Playwright
- Firecrawl
- Supabase

## Install
```bash
pip install openai-agents fastapi uvicorn playwright firecrawl-py pydantic
```
